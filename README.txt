Installation Requirements:

To set up the bot, ensure the following Python packages are installed:

pip install pytz twitchio simpleobsws
Repository Structure:

bot.py: Main script for the bot's functionality.

getuserid.py: Script for retrieving user IDs.

data/: Directory likely used for storing data related to the bot's operations.

giveawaybot/: Directory that possibly contains modules related to the giveaway functionality.

gunhtml/: Directory that might include HTML files or templates for the bot's interface.
